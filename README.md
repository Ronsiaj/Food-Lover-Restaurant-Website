# Food-Lover-Restaurant-Website
A modern single-page restaurant website built using ReactJS.

This project showcases a responsive landing page for a restaurant including sections like Home, About, Menu, Offers, Gallery, and Contact.

The website is designed with a clean UI, smooth navigation, and mobile responsiveness.

**🚀 Features**

Responsive design for desktop, tablet, and mobile

Single page navigation

Modern restaurant landing page layout

Smooth scrolling navigation

Clean and simple UI


**🛠️ Built With**

ReactJS

JavaScript (ES6)

HTML5

CSS3

HTML5

CSS3

**⚙️ Installation**

Clone the repository and install dependencies.
cd restaurant-website
npm install
npm start
